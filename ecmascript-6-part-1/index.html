ECMAScript 6 part - 1

28 December 2016 on ES6, JavaScript

ES6 is a significant update to the language, and the first update to the language since ES5 was standardized in 2009. Implementation of these features in major JavaScript engines is underway now.

Constants

Support for constants (also known as "immutable variables"), i.e., variables which cannot be re-assigned new content.

const PI = 3.141593  
PI > 3.0


Scoping

Declaration declared through var are hoisted and function scoped. Variables declared through either let or const are block-scoped and not available in the temporal dead-zone.

Blocked Scope Variables

Block-scoped variables (and constants) that should be available in blocked scope (Curly braces) should be declared using let

for (let i = 0; i < a.length; i++) {  
    let x = a[i]
    …
}
for (let i = 0; i < b.length; i++) {  
    let y = b[i]
    …
}

let callbacks = []  
for (let i = 0; i <= 2; i++) {  
    callbacks[i] = function () { return i * 2 }
}
callbacks[0]() === 0  
callbacks[1]() === 2  
callbacks[2]() === 4


Block Scoped Functions

This are clear by example

{
    function foo () { return 1 }
    foo() === 1
    {
        function foo () { return 2 }
        foo() === 2
    }
    foo() === 1
}


Arrow Functions

Expression Bodies

Old Junk looks like this 

odds  = evens.map(function (v) { return v + 1; });  
pairs = evens.map(function (v) { return { even: v, odd: v + 1 }; });  
nums  = evens.map(function (v, i) { return v + i; });

New sexy expressive closure syntax 

odds = evens.map(v => v + 1);  
pairs = evens.map(v => even: v, odd: v + 1);  
nums = evens.map( (v, i) => v + i);


Statement Bodies

Old junk example 

nums.forEach(function (v) {  
   if (v % 5 === 0)
       fives.push(v);
});

New expressive closure syntax

nums.forEach( v => {  
   if (v % 5 === 0)
      fives.push(v);
});


Lexical this

Old Junk 

var self = this;  
this.nums.forEach(function (v) {  
    if (v % 5 === 0)
        self.fives.push(v);
});

Stylish ES6 

this.nums.forEach((v) => {  
    if (v % 5 === 0)
        this.fives.push(v)
})
