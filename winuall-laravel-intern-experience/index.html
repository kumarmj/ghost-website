Winuall Intern (week 1)

In this post I will be summarising my first week experience with Winuall building backend for e-commerce platform. Winuall is an online education platform, where JEE aspirants can ask question to experts Know More.

Winuall e-commerce platform uses Laravel (PHP Web Framework) in Backend and AngularJS (JS Framework) in frontend. I am developing backend for their platform.

Laravel

Laravel is intended for the development of web applications following the model–view–controller (MVC) architectural pattern.

Eloquent ORM (object-relational mapping) is an advanced PHP implementation of the active record pattern, providing at the same time internal methods for enforcing constraints on the relationships between database objects. Following the active record pattern, Eloquent ORM presents database tables as classes, with their object instances tied to single table rows.

Migrations provide a version control system for database schemas.

Database seeding provides a way to populate database tables with selected default data.

Web API's

The server-side web API is a programmatic interface consisting of one or more publicly exposed endpoints to a defined request–response message system, expressed in JSON.

Laravel API Boilerplate is a "starter kit" you can use to build your first API.

Everyone and their dog wants an API, so you should probably learn how to build them. https://leanpub.com/build-apis-you-wont-hate
